# EC2 Basics Lab

## Goal

Practice launching and understanding a basic EC2 instance.

## Services

- Amazon EC2
- Security Groups
- Key pairs
- AMIs

## Planned Steps

1. Choose an Amazon Machine Image.
2. Select a free-tier compatible instance type.
3. Configure networking.
4. Configure a security group.
5. Launch the instance.
6. Connect with SSH if applicable.
7. Stop or terminate resources to avoid cost.

## Security Notes

- Do not open SSH to the entire internet unless there is a clear lab reason.
- Prefer restricting access to your own IP.
- Stop or terminate resources when done.
