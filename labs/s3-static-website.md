# S3 Static Website Lab

## Goal

Practice the concept of hosting static files with Amazon S3.

## Services

- Amazon S3
- Bucket policies
- Static website hosting

## Planned Steps

1. Create an S3 bucket.
2. Upload a simple `index.html` file.
3. Enable static website hosting.
4. Configure permissions safely.
5. Test the website endpoint.
6. Document what worked and what needs review.

## Security Notes

Do not upload private data, secrets, credentials, or customer information.
